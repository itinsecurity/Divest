# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

## Summary

We are committed to providing a welcoming and inclusive environment for everyone. Participants are expected to treat each other with respect and professionalism.

## Enforcement

Instances of unacceptable behaviour may be reported by opening a private GitHub issue or contacting the maintainers directly. All reports will be reviewed and investigated promptly and fairly.

For the full text of the Contributor Covenant, see:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
